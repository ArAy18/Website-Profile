# Website-Profile

Proyek website statis yang dikembangkan menggunakan HTML5 dan CSS3 dasar. Proyek ini dirancang sebagai landing page profil yang fleksibel sehingga dapat diadaptasi untuk mewakili individu, organisasi, perusahaan atau lokasi geografis tertentu dengan fokus mendemonstrasikan pemahaman kuat pada fundamental struktur web dan dokumentasi yang terstruktur.

Fitur Utama :
Struktur Semantik: Menggunakan elemen HTML sesuai fungsinya untuk SEO dan aksesibilitas.
Fundamental Styling: Penataan estetika menggunakan CSS murni tanpa framework.
Dokumentasi Logika: Dilengkapi dengan flowchart dan pseudo code untuk mempermudah pemahaman alur program.
Optimasi Desktop: Tampilan yang fungsional dan nyaman dipandang pada layar desktop.

Glosarium :
1. Clean Code (Kode Bersih) : Menulis kode dengan rapi dan terorganisir agar mudah dibaca dan dikelola oleh manusia.
2. Flowchart (Diagram Alir) : Representasi visual yang menunjukkan alur logika atau langkah-langkah kerja website.
3. Pseudo Code (Kode Semu)  : Penjelasan logika program menggunakan bahasa sehari-hari sebelum diubah menjadi kode teknis.




A static website project developed using basic HTML5 and CSS3. It is designed as a flexible profile landing page that can be adapted to represent a specific individual, organization, company, or geographic location, with a focus on demonstrating a strong understanding of web design fundamentals and structured documentation.

Key Features :
Semantic Structure  : Using HTML elements correctly for better SEO and accessibility.
Fundamental Styling : Aesthetic layout crafted with pure CSS, free from frameworks.
Logic Documentation : Includes flowcharts and pseudo code to illustrate the program's logic.
Desktop Optimized   : A functional and visually pleasing experience tailored for desktop screens.

Glossary :
Clean Code: Writing code in a neat and organized manner so it is easy for humans to read and maintain.
Flowchart: A visual representation showing the logical flow or step-by-step process of the website.
Pseudo Code: An informal way of describing program logic using plain language before translating it into technical code.
